Bem-vindo ao Fóton

Uma base sólida e elegante para o desenvolvimento de interfaces modernas.

O Fóton Framework foi desenvolvido para ser a camada visual eficiente dos seus projetos. Focamos em entregar componentes polidos, tipografia ajustada e um grid responsivo automático, permitindo que você tenha uma interface consistente desde o primeiro momento.
Nossa proposta é oferecer um design system limpo e organizado, que não exige configurações complexas. É uma ferramenta prática, desenhada para se integrar ao seu fluxo de trabalho de maneira natural e invisível.

Pilares do Fóton Framework

Leveza

Arquitetura otimizada para carregamento rápido e renderização eficiente, sem excesso de código.

Semântica

Classes nomeadas de forma lógica e intuitiva (.ft-btn, .ft-card), facilitando a leitura e manutenção.

Estética

Um visual contemporâneo e profissional, com tipografia e espaçamentos calibrados para harmonia visual.

Como Utilizar

A integração é modular. O CSS é obrigatório para o visual, e o JS é opcional para comportamentos dinâmicos.

Arquivos: Acesse a aba Download para baixar o pacote completo (CSS + JS).

1. Estrutura Recomendada

Utilize o modelo abaixo. Ele já inclui as fontes Manrope e Poppins, além do script opcional para interatividade. 

                            
    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Projeto Fóton</title>

        <!-- 1. Tipografia (Google Fonts) -->
        <link href="https://fonts.googleapis.com/css2?family=Manrope:wght@400;700;800&family=Poppins:wght@300;400;500;600&display=swap" rel="stylesheet">

        <!-- 2. Ícones (Opcional - FontAwesome) -->
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

        <!-- 3. Fóton Framework CSS -->
        <link rel="stylesheet" href="foton_framework.css">
    </head>
    <body>

        <nav class="ft-navbar">
            <div class="container">
                <a href="#" class="ft-logo">Fóton<span>App</span></a>
            </div>
        </nav>

        <main class="ft-hero">
            <div class="ft-hero-content">
                <span>Bem-vindo</span>
                <p>Sua nova interface começa aqui.</p>
                <button class="ft-btn ft-btn-gradient">Saiba Mais</button>
            </div>
        </main>

        <!-- 4. Fóton Scripts (Opcional) -->
        <!-- Adiciona interatividade: fechar alertas, menu mobile, etc -->
        <script src="foton_scripts.js"></script>
    </body>
    </html>

2. Interatividade (JavaScript)

O arquivo foton_scripts.js é opcional, mas recomendado para uma experiência completa. Ele ativa automaticamente:

  • Botões de fechar em Alertas
  • Menu Mobile (Hamburger)
  • Modais e Dropdowns
  • Animações suaves em Acordeões
  • Popover e Tooltips

3. Personalização

O Fóton utiliza Variáveis CSS nativas. Para adaptar o tema, defina os valores no seletor :root do seu arquivo de estilo: 

                            
    :root {
        /* Cor Principal */
        --primary: #820ad1; 
        
        /* Fundo Suave */
        --primary-soft: #f5eafe; 
        
        /* Tema Escuro */
        --dark: #111111;
    }
