Bem-vindo ao Fóton

O framework CSS essencial para desenvolvedores que buscam performance e estética.

O Fóton Framework não é apenas mais uma biblioteca CSS. Ele foi projetado para ser a base invisível de interfaces de alta conversão. Enquanto outros frameworks focam em ter milhares de componentes que você nunca vai usar, o Fóton foca no essencial: tipografia perfeita, espaçamento rítmico e componentes que vendem.

Por que escolher o Fóton?

Ultra Leve

Sem JavaScript pesado. Sem dependências. Apenas CSS moderno e otimizado que carrega instantaneamente.

Layout Adaptável

Esqueça media queries complexas. Nosso Grid System inteligente se adapta a qualquer tela automaticamente.

Focado em Design

Variáveis CSS globais permitem que você mude a identidade visual inteira alterando apenas algumas cores.

Começando Agora

Integrar o Fóton ao seu projeto é simples. Você só precisa de dois passos.

Dica: Certifique-se de baixar a última versão do arquivo foton_framework.css na aba Download.

1. Estrutura Básica

Copie o template abaixo para iniciar seu projeto com todas as configurações necessárias, incluindo as fontes Manrope e Poppins.

<!DOCTYPE html> <html lang="pt-br"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meu Projeto Fóton</title> <!-- 1. Fontes Google (Manrope + Poppins) --> <link href="https://fonts.googleapis.com/css2?family=Manrope:wght@200;300;400;500;600;700;800&family=Poppins:wght@100;200;300;400;500;600;700;800;900&display=swap" rel="stylesheet"> <!-- 2. Ícones (Opcional - FontAwesome) --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css"> <!-- 3. Fóton Framework CSS --> <link rel="stylesheet" href="foton_framework.css"> </head> <body> <nav class="navbar"> <div class="container"> <a href="#" class="logo">Meu<span>App</span></a> </div> </nav> <main class="hero"> <div class="hero-content"> <span>Olá, Mundo!</span> <p>Seu projeto Fóton está rodando.</p> <button class="btn btn-gradient">Vamos lá</button> </div> </main> <!-- Scripts de Interatividade (Opcional) --> <script src="foton_scripts.js"></script> </body> </html>

2. Customização Rápida

Para deixar o framework com a cara da sua marca, abra o arquivo CSS e edite as variáveis no topo (:root).

:root { /* Mude a cor principal aqui */ --primary: #40aa54; /* Ajuste o modo escuro */ --dark: #1d1e25; }
