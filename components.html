Biblioteca de Componentes

Documentação técnica dos elementos de interface do Fóton Framework.

Botões

Utilize a classe base .btn seguida de um modificador de estilo. A tag pode ser <button> ou <a>. Os botões possuem transições suaves, estados de hover automáticos e reset de estilos nativos. 

<!-- Botão com gradiente da marca (Destaque Principal) -->
<button class="ft-btn ft-btn-gradient">Ação Principal</button>

<!-- Botão secundário/neutro (Fundo cinza claro) -->
<button class="ft-btn ft-btn-primary">Cancelar</button>

<!-- Botão sólido escuro (Contraste alto) -->
<button class="ft-btn ft-btn-dark">Dark</button>

Botões Semânticos

Use cores semânticas para indicar o feedback ou propósito da ação (sucesso, aviso, perigo, informação). Adicione a classe correspondente ao estado desejado.

<!-- Classes Semânticas -->
<button class="ft-btn ft-btn-success">Success</button>
<button class="ft-btn ft-btn-warning">Warning</button>
<button class="ft-btn ft-btn-danger">Danger</button>
<button class="ft-btn ft-btn-info">Info</button>

Botões Outline (Borda)

Para interfaces mais leves e secundárias, use as variações .btn-outline-*. Inclui variantes translúcidas (White/Dark) para uso sobre fundos coloridos ou imagens.

<!-- Outlines Coloridos -->
<button class="ft-btn ft-btn-outline-success">Success</button>
<button class="ft-btn ft-btn-outline-warning">Warning</button>
<button class="ft-btn ft-btn-outline-danger">Danger</button>
<button class="ft-btn ft-btn-outline-info">Info</button>

<!-- Translúcidos (Fundo Transparente) -->
<button class="ft-btn ft-btn-outline-dark">Dark Outline</button>
<button class="ft-btn ft-btn-outline-white">White Outline</button>

Links e Tamanhos

Adicione .ft-btn-sm para botões compactos (padding reduzido e fonte menor) ou .ft-btn-block para ocupar 100% da largura do container pai (display flex). Para links que parecem texto mas têm comportamento de botão, use .ft-btn-link.

Link (.ft-btn-link) Botão Pequeno (.ft-btn-sm) Botão Bloco (.ft-btn-block) 
<!-- Tamanhos -->
<a href="#" class="ft-btn ft-btn-sm ft-btn-dark">Pequeno</a>
<a href="#" class="ft-btn ft-btn-link">Estilo Link</a>

<!-- Bloco (Full Width) -->
<a href="#" class="ft-btn ft-btn-block ft-btn-primary">Botão de Bloco</a>

Dropdowns

Menus flutuantes acionáveis. O framework suporta ft-dropdowns via CSS (hover) mas recomenda-se o uso do atributo data-toggle="ft-popover" no botão gatilho para melhor suporte em dispositivos touch e controle via JavaScript (clicar fora para fechar). A estrutura exige um container .ft-dropdown envolvendo o botão e a lista .ft-dropdown-menu.

    <!-- Gatilho com ícone -->
    <button class="ft-btn ft-btn-primary" data-toggle="ft-popover">
        Menu <i class="fa-solid fa-chevron-down"></i>
    </button>

    <!-- Lista de Itens -->
    <ul class="ft-dropdown-menu">
        <li><a class="ft-dropdown-item" href="#">Item 1</a></li>

        <li class="ft-dropdown-div"></li>

    </ul>
</div>

Popovers & Tooltips

Elementos flutuantes para informações contextuais. Tooltips são ativados via hover e implementados com CSS puro usando o atributo data-tooltip. Popovers são ativados via clique e requerem a estrutura HTML completa e o atributo data-toggle="ft-popover" no botão para que o JavaScript gerencie a abertura e o fechamento ao clicar fora.

Toggle Switch

Um checkbox estilizado como interruptor físico. A estrutura requer um <label> envolvendo o input checkbox oculto e uma div .ft-toggle-track que renderiza o visual. 

<label class="ft-toggle">
    <input type="checkbox">
    <div class="ft-toggle-track"><span></span></div>
    <span>Label do Switch</span>
</label>

Alertas

Componentes de feedback para mensagens de usuário. Utilize a classe .ft-alert combinada com uma cor contextual (ex: .ft-alert-warning). O botão de fechar requer a classe .ft-alert-close e a funcionalidade de "dismiss" é gerenciada automaticamente pelo script foton_scripts.js.

Operação Realizada Seus dados foram salvos com sucesso no banco de dados.
Atenção: Sua assinatura expira em 3 dias.
Erro Crítico: Falha ao conectar ao servidor.
Info: Nova versão disponível para download.
Info: Olá esse é um ft-alert-neutral.
<div class="ft-alert ft-alert-primary">
    <!-- Ícone opcional -->
    <i class="fa-solid fa-check ft-alert-icon"></i>

    <!-- Conteúdo -->
    <div>
        <span class="ft-alert-heading">Título</span>
        Mensagem de corpo.
    </div>

    <!-- Botão de fechar (JS automático) -->
    <button class="ft-alert-close"><i class="fa-solid fa-xmark"></i></button>
</div>

Badges

Indicadores visuais para status, contadores ou categorias. A classe base .ft-badge aplica padding e alinhamento. Combine com cores contextuais (ex: .ft-badge-success) para alterar o fundo e a cor do texto. Adicione .ft-badge-pill para bordas totalmente arredondadas (estilo pílula).

<!-- Estilo Pill (Totalmente arredondado) -->

Use as classes de tamanho para ajustar as dimensões: .ft-avatar-sm (32px), .ft-avatar-md (48px), .ft-avatar-lg (64px) ou .ft-avatar-xl (96px). Para agrupar ft-avatares, envolva-os em um container .ft-avatar-group.

Acordeões

Componente expansível para FAQs ou conteúdo colapsável. Baseado na tag nativa <details> e <summary> para máxima semântica e acessibilidade. O arquivo JavaScript do framework adiciona automaticamente uma animação suave de altura (slide down/up) ao abrir e fechar, garantindo uma experiência fluida (60fps).

Como funciona a animação?
O JavaScript calcula a altura do conteúdo e transiciona a propriedade height, oferecendo uma experiência de 60fps.
É necessário JavaScript?
Sim, para a animação. Sem ele, funciona com toggle instantâneo. 
<details class="ft-accordion">
    <summary>Pergunta ou Título</summary>
    <div class="ft-accordion-body">
        Conteúdo oculto que será expandido...
    </div>
</details>

Modais

Janelas de sobreposição para conteúdo focado. Requer estrutura específica e atributos de data para funcionamento via JavaScript.

Título do Modal

Este é o conteúdo do corpo do modal. Você pode colocar texto, formulários ou qualquer outro elemento HTML aqui.

<!-- Botão Gatilho (data-target deve corresponder ao ID) -->
<button class="ft-btn" data-toggle="modal" data-target="meuModal">
    Abrir Modal
</button>

<!-- Estrutura do Modal -->
<div class="ft-modal" id="meuModal">
  <div class="ft-modal-dialog">
    <div class="ft-modal-content">
    
      <div class="ft-modal-header">
        <h4 class="ft-modal-title">Título do Modal</h4>
        <button class="ft-close" data-dismiss="modal">×</button>
      </div>
      
      <div class="ft-modal-body">
        <p>Conteúdo do modal aqui...</p>
      </div>
      
      <div class="ft-modal-footer">
        <button class="ft-btn ft-btn-sm" data-dismiss="modal">Fechar</button>
        <button class="ft-btn ft-btn-sm ft-btn-primary">Salvar</button>
      </div>
      
    </div>
  </div>

Componentes estruturais para montar o layout macro do seu site.

Top Bar

Uma barra utilitária fina localizada acima da navegação principal. Ideal para anúncios, promoções ou links secundários. Utilize a classe .ft-top-bar.

Hero Section

Área de destaque inicial. Possui padding generoso (padding-top ajustado para compensar a ft-navbar fixa) e centralização vertical/horizontal do conteúdo dentro de .ft-hero-content.
