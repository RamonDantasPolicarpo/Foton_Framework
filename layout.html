Layout & Grid

Classes utilitárias para estruturação de conteúdo responsivo.

Grid Automático

A classe .grid-responsive utiliza CSS Grid com auto-fit e minmax para criar colunas que se adaptam automaticamente ao espaço disponível, sem necessidade de Media Queries manuais.

Coluna 2
Coluna 3
<div class="grid-responsive"> <div>Conteúdo 1</div> <div>Conteúdo 2</div> <div>Conteúdo 3</div> </div>

Flex Container

A classe .flex-container utiliza Flexbox com wrap, ideal para listas de itens centralizados, galerias de ícones ou cards de funcionalidades.

Flex Item
Flex Item
Flex Item
<div class="flex-container"> <!-- Seus elementos com largura definida --> <div style="width: 200px;">Item 1</div> <div style="width: 200px;">Item 2</div> </div>

Seções de Destaque

A classe .section-dark cria um container escuro com bordas arredondadas e padding generoso, ideal para quebrar o ritmo da página ou destacar features importantes. Ela já centraliza o conteúdo e limita a largura.

Título da Seção

Subtítulo descritivo em cor clara.

<section class="section-dark"> <div class="section-header"> <h2>Título</h2> <p>Descrição...</p> </div> <!-- Conteúdo da seção --> </section>

Layout de Formulários

Aprenda a estruturar seus formulários utilizando CSS Grid e Flexbox nativos junto com as classes do framework.

1. Pilha Vertical (Padrão)

Os elementos de input nativos possuem largura automática. Ao adicionar a classe .form-input, eles expandem para 100% da largura do container pai, ideal para layouts móveis ou empilhados.

<!-- Container .form-group agrupa label e input --> <div class="form-group"> <label class="form-label">Email</label> <input type="email" class="form-input"> </div>

2. Grid de Colunas (Lado a Lado)

Para colocar campos lado a lado (ex: Nome e Sobrenome), envolva-os em uma div com display: grid. Você pode definir as colunas manualmente.

<!-- Grid de 2 colunas iguais --> <div style="display: grid; grid-template-columns: 1fr 1fr; gap: 20px;"> <div class="form-group">...</div> <div class="form-group">...</div> </div>

3. Layout Inline (Botão ao lado)

Use display: flex para alinhar um campo de input com um botão. Use align-items: flex-end para alinhar o botão com o input (ignorando o label).

<div style="display: flex; gap: 10px; align-items: flex-end;"> <div class="form-group" style="flex-grow: 1;"> <label class="form-label">Cupom</label> <input type="text" class="form-input"> </div> <button class="btn btn-dark">Aplicar</button> </div>
