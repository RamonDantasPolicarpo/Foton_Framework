Layout & Grid

Classes utilitárias para estruturação de conteúdo responsivo.

Grid de Preços (Auto-Responsive)

A classe .grid-responsive utiliza CSS Grid com minmax para criar colunas que se adaptam automaticamente ao espaço disponível.
