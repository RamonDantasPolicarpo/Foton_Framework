Formulários

Componentes de input e busca.

Barra de Busca

O .input-group ocupa 100% do container pai por padrão. Ajuste a largura usando CSS inline ou classes utilitárias se precisar.

Padrão (100%)

Largura Personalizada (ex: 400px)

<!-- Input Group --> <div class="input-group"> <i class="fa-solid fa-search" style="margin-left: 15px;"></i> <input type="text" class="input-field" placeholder="Buscar..."> <button class="btn btn-gradient">Pesquisar</button> </div>

Inputs Básicos

Campos de texto padrão, email e senha. Utilize a classe .form-input para expandir e estilizar.

<div class="form-group"> <label class="form-label">Email</label> <input type="email" class="form-input" placeholder="..."> </div>

Seleção e Opções

Selects, checkboxes e radio buttons customizados. A fonte das opções agora herda corretamente.

<!-- Select --> <select class="form-input"> <option>Opção 1</option> <option>Opção 2</option> </select> <!-- Checkbox --> <label class="form-check"> <input type="checkbox" class="form-check-input"> <span class="form-check-label">Texto</span> </label> <!-- Radio --> <label class="form-check"> <input type="radio" name="radioName" class="form-check-input"> <span class="form-check-label">Opção</span> </label>

Áreas de Texto

<div class="form-group"> <label class="form-label">Mensagem</label> <textarea class="form-input" rows="4"></textarea> </div>
